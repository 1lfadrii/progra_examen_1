# progra_examen_1
# Gestion de usuarios de correo
Aplicación de consola en java para registrar y listar usuarios

## funcionalidades
Registrar usuario (nombre y correo)
Listar usuarios registrados
Guardar datos en archivo usuarios.txt

## tecnologías usadas
Arquitectura por capas
Manejo de archivos
try with resources
try catch
Validaciones básicas

## Como ejecutarlo
1. Clonar el repositorio
2. Abrir en netbeans
3. Ejecutar la clase App.java
