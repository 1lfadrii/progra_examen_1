Gestion de usuarios de correo
Aplicación de consola en java para registrar y listar usuarios

funcionalidades
Registrar usuario (nombre y correo) Listar usuarios registrados Guardar datos en archivo usuarios.txt

tecnologías usadas
Arquitectura por capas Manejo de archivos try with resources try catch Validaciones básicas

Como ejecutarlo
Clonar el repositorio
Abrir en netbeans
Ejecutar la clase App.java
